Software in Python

control-theory-plot

View on Github

Graphical interface for plotting the usual graphs in control theory (step response, Bode plot, Root Locus and Nyquist plot).

The transfer function can be either in the num/den format or in the zpk format. The closed-loop can be compensated with either a gain, a PID or even a lag-lead.

industrial-networks

View on Github

A collection of four scripts emulating some communication protocols used in automation industry: Serial, TCP/IP, CAN and Modbus (done during a practical class).

numerical-methods

View on Github

A collection of four scripts implementing the most basic numerical methods: nonlinear equations, polynomial interpolation, ordinary differential equation, and stationary iterative methods (done during a practical class).

Applications Python

control-theory-plot

Voir sur Github

Interface graphique générant les graphes usuels en asservissement continu (réponse indicielle, digramme de Bode, Lieu d'Evans et Lieu de Nyquist).

La fonction de transfert est entrée soit sous la forme num/den ou bien la forme zpk. La boucle fermée peut être régulée par un gain, un PID ou un avance-retard de phase.
