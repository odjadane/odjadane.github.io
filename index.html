Applications Python

control-theory-plot

Voir sur Github

Interface graphique générant les graphes usuels en asservissement continu (réponse indicielle, digramme de Bode, Lieu d'Evans et Lieu de Nyquist).

La fonction de transfert est entrée soit sous la forme num/den ou bien la forme zpk. La boucle fermée peut être régulée par un gain, un PID ou un avance-retard de phase.

serial-communication-diagram

Voir sur Github

Interface graphique générant l'oscillogramme d'une communication série asynchrone transmettant un caractère, selon les propriétés de la trame choisies par l'utilisateur.

J'ai écrit ce programme pour me faciliter le débogage du port série d'un microcontrôleur.
